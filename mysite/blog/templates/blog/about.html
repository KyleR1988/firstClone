Thanks for checking it out

This is by first attempt at a blog clone project The website is built using Django 2.1, Bootstrap 4.1.3 and Python 3.7.1

The build took around 3 weeks to complete as there were a few errors. Mostly just simple spelling mistakes. I also had to adapt my clone to Bootstarp 4.1.3 as the course from where I copied the project used bootstrap 3.3.7. The major differnce was around the navbar, making sure it collapsed when and how it was suppoesed to. Also the icons in my project are not Glyphicons, I used Font Awesome as per the the Bootstrap Documnetation Suggestion

The way the website works is as follows:
  1. The owner of the website can login and creat a new post
  2. The post can then be edited, trashed or published
  3. The owner can add users to the site from the admin page
  4. Users can add posts and comment on the posts available
  5. Only users can apporve or delete comments on posts

All the comments and posts are stored in database tables. The users can access these tables through the admin login provided to them. From there, they can add othre users, add comments, delete posts and comments or approve posts and comments

Some other cool items I used were the Medium website style editor for input text. This I got from: MEDIUM EDITOR. The color changing items I got from CodePen.
